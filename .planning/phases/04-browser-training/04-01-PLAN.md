STEP 1 — Add _ReplayEnv class inside train_browser(), after the _FakeEnv class definition (around line 278). _ReplayEnv replays collected (obs, reward, done) tuples so SB3's learn() machinery can process them:

```
class _ReplayEnv(gym.Env):
    """Env that replays browser-collected (obs, reward, done) transitions for SB3 learn()."""
    observation_space = spaces.Box(low=-1.0, high=1.0, shape=(OBS_DIM,), dtype=np.float32)
    action_space = spaces.Discrete(N_ACTIONS_GEN9)

    def __init__(self, transitions: list) -> None:
        super().__init__()
        self._transitions = list(transitions)
        self._idx = 0

    def reset(self, **kwargs):
        self._idx = 0
        return np.zeros(OBS_DIM, dtype=np.float32), {}

    def step(self, action):
        if self._idx < len(self._transitions):
            obs, reward, done = self._transitions[self._idx]
            self._idx += 1
        else:
            obs = np.zeros(OBS_DIM, dtype=np.float32)
            reward = 0.0
            done = True
        return obs, reward, done, False, {}
```

STEP 2 — Collect transitions during the battle loop. At the point where obs is extracted and an action is chosen (around line 353-355), store the transition. The reward for each intermediate step is 0.0 (already being appended to rewards list); the per-step transition is: `(obs, 0.0, False)`. For the terminal step, replay the final reward.

Add a `transitions: list[tuple] = []` list at the start of each battle (alongside `rewards: list[float] = []`).

In the per-turn block (after `obs = build_observation_from_dom(page1)` and `_pick_move_from_obs(page1, obs, policy)`), append:
  `transitions.append((obs.copy(), 0.0, False))`

When the battle ends with a win/loss result (where `reward = 1.0 if "win" ...`), before the `break`, update the last transition with the terminal reward and done=True by appending:
  `transitions.append((build_observation_from_dom(page1), reward, True))`

STEP 3 — Call policy.learn() at each checkpoint interval. In the periodic checkpoint block (around line 374), after saving the checkpoint zip files, add the learn() call:

```python
if policy is not None and SB3_OK and steps - last_swap >= swap_every:

    policy.save(str(ckpt_path))
    import shutil
```
